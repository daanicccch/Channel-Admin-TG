# Шаблоны постов для Crypto Digest Bot

## Дайджест (утро/вечер)

**Длина:** 800-1500 символов

**Структура:**
1. Эмодзи + **жирный заголовок** (например: "Утренний крипто-дайджест" или "Вечерний обзор рынка")
2. 3-5 блоков, каждый:
   - Эмодзи + заголовок темы жирным
   - 1-2 предложения с ключевыми фактами и цифрами
3. Заключение или вопрос к аудитории
4. Хештеги (3-5 штук)

**Пример структуры:**
```
🌅 **Утренний крипто-дайджест | 15 марта**

🔹 **SOL обновил максимумы**
Solana достигла $180, рост за сутки составил 8.5%. Объём торгов вырос на 40%.

🔹 **Новый DEX на Solana**
Протокол XYZ запустился с TVL $50M в первые 24 часа.

🔹 **Bitcoin стабилен**
BTC торгуется в диапазоне $67-68K, доминация — 52%.

Какая новость больше всего удивила? Пишите в комментариях 👇

#crypto #solana #bitcoin #дайджест
```

## Аналитика

**Длина:** 1500-3000 символов
**Обязательно:** изображение (график, инфографика)

**Структура:**
1. Эмодзи + **жирный заголовок** (аналитическая тема)
2. Постановка проблемы/вопроса
3. Данные и цифры (метрики, сравнения, графики)
4. Анализ — что означают эти данные
5. Вывод — к чему это может привести
6. Источники данных

**Пример структуры:**
```
📊 **Анализ: Solana DeFi растёт быстрее Ethereum?**

Последние 30 дней показали интересную динамику в DeFi-секторе Solana.

📈 **Цифры:**
- TVL Solana DeFi: $8.2B (+25% за месяц)
- Количество активных кошельков: 2.1M (+15%)
- Средний TPS: 3,500

🔍 **Что это значит:**
Рост TVL опережает динамику цены SOL, что может указывать на органический приток капитала...

💡 **Вывод:**
При сохранении текущей динамики Solana может...

Источники: DeFiLlama, Dune Analytics, Solscan
```

## Алерт

**Длина:** 300-800 символов

**Структура:**
1. ⚡ **ALERT** + жирный заголовок (суть события)
2. Ключевой факт + цифры
3. Что это значит для рынка/пользователей
4. NFA/DYOR дисклеймер

**Пример структуры:**
```
⚡ **ALERT: Jupiter объявил о раздаче $200M в JUP**

Крупнейший агрегатор на Solana анонсировал второй раунд аирдропа. Объём — $200M в токенах JUP.

Что это значит: ожидается повышенная активность в экосистеме Solana, возможен рост TVL и объёмов.

⚠️ NFA / DYOR — не является финансовым советом, проводите собственный анализ.
```

## Недельный дайджест

**Длина:** 2000-4000 символов

**Структура:**
1. Эмодзи + **жирный "Итоги недели"** + даты
2. 5-7 ключевых событий (каждое — эмодзи + заголовок + 1-2 предложения)
3. Рыночная статистика (таблица или список с цифрами)
4. Прогноз/взгляд на следующую неделю
5. Вопрос-опрос для аудитории

**Пример структуры:**
```
📅 **Итоги недели | 10-16 марта**

🔥 **Главные события:**

1️⃣ **SOL пробил $200**
Впервые с 2021 года Solana преодолела психологическую отметку...

2️⃣ **Ethereum Dencun обновление**
Успешная активация привела к снижению комиссий L2 на 90%...

3️⃣ **SEC одобрила новые ETF**
...

📊 **Рынок в цифрах:**
- BTC: $68,500 (+3.2%)
- ETH: $3,800 (+5.1%)
- SOL: $205 (+12.4%)
- Общая капитализация: $2.65T
- Индекс страха/жадности: 72 (Жадность)

🔮 **Взгляд на следующую неделю:**
На горизонте — заседание ФРС и данные по инфляции...

Какое событие было самым важным на этой неделе?
1. SOL $200
2. Dencun обновление
3. SEC и ETF

#crypto #итогинедели #solana #bitcoin
```
