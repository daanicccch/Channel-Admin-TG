# Шаблоны постов для ChannelBot

## Post

Используй как формат по умолчанию для `/post`.

Правила:
- Держи объём примерно на уровне исходного поста.
- Не растягивай короткий источник в лонгрид.
- 1 заголовок + 1-3 коротких абзаца.
- Если в источнике один главный тезис, в посте тоже должен быть один главный тезис.

Скелет:
```text
💎 <b>Короткий заголовок по сути</b>

1 абзац: что произошло.

2 абзац: почему на это смотрят / что уже известно.

Короткий вывод или watchline без воды.
```

## Alert

Используй для `/post_alert`.

Правила:
- Самый короткий и самый быстрый формат.
- Один триггер, одна суть, минимум фона.
- Никаких длинных рассуждений и распаковок.

Скелет:
```text
⚡ <b>ALERT: что произошло</b>

Ключевой факт в 1-2 предложениях.

Почему это заметили / что важно отслеживать дальше.
```

## Weekly

Используй для `/post_weekly`.

Правила:
- 2-4 события недели, только действительно сильные.
- Не делай weekly из одного слабого инфоповода, если данных мало.
- Каждый блок короткий, без лишней аналитики.

Скелет:
```text
🗓 <b>Итоги недели</b>

1. Событие недели + 1-2 предложения.
2. Событие недели + 1-2 предложения.
3. Событие недели + 1-2 предложения.

Финальный короткий вывод по неделе.
```
