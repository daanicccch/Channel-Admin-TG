# Humanizer Rules

Goal: turn an AI draft into a post that reads like a live crypto editor wrote it.

## Keep
- Facts, numbers, names, links, tickers, hashtags, and HTML tags.
- The main meaning of every block.
- The same topical focus as the source material.

## Remove
- Office language.
- Press-release tone.
- Empty transitions.
- Vague authority references.
- Generic hype words.

## Rewrite toward
- Direct wording.
- Concrete verbs.
- Shorter phrasing where possible.
- Human rhythm instead of equal-length sentences.
- One grounded editor observation when it follows from the data.

## Final check
- Does it sound like a Telegram post?
- Is every strong phrase backed by a fact?
- Did any fake certainty or extra fact slip in?
